Tamaño flexible de los elementos

Las flexboxes también pueden adaptarse en tamaño. Para eso, utilizamos el comando flex. De forma parecida al orden, los elementos pueden dividirse en grupos; cuanto mayor es el valor, más espacio ocupa el elemento.

De hecho, flex es una forma abreviada del comando, que contiene tres instrucciones: flex-grow, flex-shrink y flex-basis. Así, puedes introducir los valores individuales directamente en forma abreviada (flex: flex-grow, flex-shrink flex-basis y flex: 1 1 20em) o determinar un solo valor y dejar que CSS se encargue de la configuración.
